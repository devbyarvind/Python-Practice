# Python Programs Collection

This repository contains various Python programs that cover fundamental concepts in Python programming, ranging from basic arithmetic operations to conditional statements, loops, and functions.
## 2026-02-01 Programs
### 1. **Swap_two_variables.py**
   - **Description**: This script swaps the values of two variables without using a third variable.
   - **Concepts**: Variable assignment, tuple unpacking, and basic input/output.

### 2. **Generate_random_number.py**
   - **Description**: Generates a random number within a specified range.
   - **Concepts**: Random number generation using Python's `random` module.

### 3. **Kilometer_to_miles.py**
   - **Description**: Converts a given distance in kilometers to miles.
   - **Concepts**: Unit conversion, simple arithmetic operations.

### 4. **Add_two_numbers.py**
   - **Description**: Adds two numbers provided by the user and displays the result.
   - **Concepts**: User input, arithmetic operations, basic input/output.

### 5. **Check_even_odd.py**
   - **Description**: Checks if the given number is even or odd.
   - **Concepts**: Conditional statements (`if-else`), modulus operator.

### 6. **Positive_negative_zero.py**
   - **Description**: Determines if the given number is positive, negative, or zero.
   - **Concepts**: Conditional statements (`if-elif-else`), basic input/output.

### 7. **Largest_of_3_numbers.py**
   - **Description**: Finds the largest of three numbers entered by the user.
   - **Concepts**: Conditional statements, comparisons, and input handling.

### 8. **Display_multiplication_table.py**
   - **Description**: Displays the multiplication table of a given number.
   - **Concepts**: Loops (`for`), basic arithmetic operations.

### 9. **Reverse_number.py**
   - **Description**: Reverses a given number.
   - **Concepts**: String manipulation, loops.
